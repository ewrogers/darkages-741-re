## Purpose

The client sends this message for **login**.

`net_send_login_request` writes opcode `0x03` as body byte 0 and reaches `net_submit_client_packet`. After submission, it copies the name argument into `session_character_name`. Session setup later uses that same name to build the MD5 salt source for derived-key packets.
