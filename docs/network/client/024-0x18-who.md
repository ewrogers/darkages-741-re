# Who (`CWho`)

| Item | Value |
| --- | --- |
| Direction | Client to server |
| Command | `0x18` (24) |
| Encoding | session key |
| Name provenance | Project protocol vocabulary, confirmed against the local builder and user-list UI owner. |

## Purpose

The client sends `CWho` to request the current world-user list. A user-list control owned by `GUIBackPane` reaches this builder.

## Body

```text
packet CWho {
    u8 opcode                    // 0x18
}
```

`net_send_who` submits exactly this one-byte plaintext body.

## Response

The server answers with [`SShowUsers`](../server/054-0x36-show-users.md). That response replaces the pane's current row data and opens or reveals the user-list pane.
