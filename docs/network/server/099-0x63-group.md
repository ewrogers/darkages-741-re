## Purpose

The server sends this message for **group**.

The constructor calls `net_server_packet_base_ctor` with opcode `0x63` and installs the `SGroup` vtable. `net_server_packet_factory_ctor` registers the same opcode with this constructor.

The gameplay handler reaches two client-side paths. With local `GroupAnswer` disabled, it constructs the normal group prompt. With the setting enabled, it skips that dialog and immediately sends [`CGroup`](../client/046-0x2e-group.md) action `3` using the supplied user text.
