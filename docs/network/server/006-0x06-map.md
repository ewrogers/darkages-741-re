## Client handling

The dispatcher reaches `map_handle_s_map_row` at `0x5F2A60` for row-oriented map data. The handler reads a `u16be` row index and three `u16be` tile identifiers for every map column. The final row triggers map CRC calculation, local cache writing, and attachment to the world pane. See [Map loading and rendering](../../map/loading-and-rendering.md).

The relationship between the class's four leading byte fields and the row data view is still being reconstructed, so the preliminary model above remains intentionally neutral.
