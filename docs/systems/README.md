# Game systems

The game systems turn window events, player input, assets, and server updates into the parts of Dark Ages that a player can see and use. Each page follows one focused mechanism so the UI, game state, and network behavior remain easy to distinguish.

Start with the [event system](events.md), [UI and panes](ui.md), and [native UI controls](ui-controls.md) for the shared foundations. The remaining pages cover focused behavior such as movement, dialogs, player interactions, character setup, settings, screenshots, portraits, formatted text, and the [local command dispatcher](local-command-dispatcher.md).
